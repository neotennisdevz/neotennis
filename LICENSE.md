## 🛡️ Playing Smart — Tips for Safer Use

Responsible use of any scripting tool starts with smart configuration choices. Keep these guidelines in mind:

1. **Always enable `LegitMode = true`** — This introduces randomized micro-delays between inputs, making automation patterns less detectable

3. **Keep `SpeedMultiplier` below 1.8** — Values above 2.0 create visual anomalies that other players and moderation systems can notice

5. **Use `RandomDelay = {0.08, 0.22}`** — A slightly wider window than the default adds natural variance
6. **Update frequently** — Outdated scripts may trigger detection after game patches; always pull the latest release from this repo
7. **Don't script in official tournaments** — Use these tools for casual and practice modes to keep competitive integrity intact

## 🔨 Troubleshooting & Common Issues

Having trouble? Work through these solutions before opening a GitHub Issue:

### ❌ Script Doesn't Load After Execute

- Verify your injector is fully attached before executing (look for the green "injected" status indicator)
- Ensure you are using the **latest version** of the script from this repository
- Try re-joining the game and re-injecting from scratch

### ❌ GUI Appears But Features Don't Activate

- Check that the game has fully loaded before executing — wait for the lobby screen
- Disable any conflicting scripts running simultaneously
- Reset config values to defaults and re-execute

### ❌ Auto Hit Fires at Wrong Timing

- Update to the latest version (timing detection is patched with each game update)
- Confirm that `LegitMode` is not set to an excessively high delay value
- Check `RandomDelay` — values above `{0.3, 0.6}` can cause timing drift

### ❌ Mobile GUI is Not Visible

- Enable overlay permissions for the injector in your device's app settings
- For Android: `Settings > Apps > [Injector] > Display over other apps > Allow`
- For iOS: Re-trust the developer certificate and relaunch both apps

### ❌ Ball Trajectory Lines Appear Laggy

- Lower your `AimLength` config value
- Disable `PlayerESP` if running simultaneously — it competes for render priority
- Ensure your device meets recommended (not just minimum) specifications

### ❌ macOS Injection Fails

- Check that your macOS injector version matches your current macOS build number
- Run the injector as administrator via Terminal: `sudo ./injector`
- Temporarily disable FileVault during the injection step if experiencing permission blocks
